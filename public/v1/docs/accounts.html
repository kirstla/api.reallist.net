Account objects allow you to create multiple accounts and add, edit and delete transactions related to the account, while tracking the accounts balance.

How to add a new Account

				POST http://api.reallist.net/v1/accounts
				
				{
					"name": "Account Name",
					"limit": 150,
					"balance": 100.00,
					"bank": {
						"name": "Bank Name",
						"routing_number": 1234567890,
						"account_number": 1234567890
					}
				}
Account Number
This is an account number that the user can define, different from the bank account number, such as a lease account number, client account number etc.
				201 Created
				Location: http://api.reallist.net/v1/accounts/a1s2d3f4g5
				
				{
					"id": "a1s2d3f4g5",
					"balance": 100.00,
					"bank": {
						"name": "Bank Name",
						"routing_number": 1234567890,
						"account_number": 1234567890
					},
					"limit": 150,
					"name": "Account Name",
					"transactions": [
						{
							"id": "0l9k8j7h6g5f4d3s2a1",
							"timestamp": 1234567890.00,
							"amount": 100.00,
							"content" "Initial Balance"
						}
					]
				}

Possible Errors

  • Not Allowed
  • Invalid
  • Required Field is empty

How to add a Transaction to an Account

				POST http://api.reallist.net/v1/accounts/a1s2d3f4g5/transactions
				
				{
					"timestamp": 1234567890.00,
					"amount": 10.00,
					"content" "New Deposit"
				}

The transaction will be added and the account's balance will be recalculated.

				201 Created
				Location: http://api.reallist.net/v1/accounts/a1s2d3f4g5/transactions/0l9k8j7h6g5f4d3s2a1

					"id": "0l9k8j7h6g5f4d3s2a1",
