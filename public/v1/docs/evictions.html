Evictions

With Eviction objects you can manage eviction cases against tenants. You can create, update and delete eviction cases, and retrieve individual evictions as well as a list of all evictions. You can also change the status of the eviction.

If you need to evict a tenant, you can create an eviction using the tenants lease id, and attach all necessary paper work, add events and expiration dates for the events.
