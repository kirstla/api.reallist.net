A property is a piece of real estate, could be land, or a building on land. Each property is made of lots, buildings, units, floors, rooms, entrances and windows. It is a written representation of your property and what makes up the property. You can create your property by defining the details of each component, such as size, color, build date, coordinates and more.

How to add a new Property

				POST http://api.reallist.net/v1/properties
				
				{
					"name": "Property Name"
				}
			
				201 Created
				Location: http://api.reallist.net/v1/properties/a1s2d3f4g5
				
				{
					"id": "a1s2d3f4g5",
					"name": "Property Name",
					"lots": []
				}

Possible Errors

  • Not Allowed
  • Invalid

Retrieves the property using the properties unique Id.
